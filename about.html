About pew

pew is an expressive mini-language designed around terminal-first math, symbolic calculus, simple games, and direct downloads.

Design philosophy

pew blends three core ideas:

  • Expressive shell syntax: short commands like dy/dx(...), int(...), and prnt make interaction fast.
  • Powerful math: symbolic derivatives, integrals, polynomial factoring, and differential equation workflows are built into the language.
  • Playful interface: text games, ASCII animations, and a colorful terminal experience keep the language engaging.

Key features

  • Symbolic derivatives and integrals for common expressions.
  • Numerical definite integrals and polynomial solvers.
  • Mini game API with sprites and movement commands.
  • Reusable shell-style commands and translation dictionaries.
  • Rich website with PHP-powered download interface and source preview.

Future direction

pew is evolving toward a universal, modern shell language with:

  • better parser support for multi-letter identifiers and functions
  • expanded calculus with integral rules and differential equation solvers
  • built-in package and module support
  • cross-platform native builds and installer packages
